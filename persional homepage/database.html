Remote DataBase System

A DataBase System using MySQL and SFML library

Introduction

This is a remote database on a web server built based on Apache. The database can be operated using MySQL through the server remotely, and only the authorized person can have access to it.

Instruction

1. Start an Apache Service

2. Start a MySQL Server

3. Open the webpage through the link localhost/fcao.html

4. Enter the information that is indicated on the web page to retrieve needed records
